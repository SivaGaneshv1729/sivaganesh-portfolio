# Portfolio Website

A personal portfolio website showcasing software development projects, skills, and professional experience.

## Features
- Modern, responsive design with dark/light mode toggle.
- Interactive project showcase with a scrolling visual track.
- Dynamic typing animation in the hero section.
- Cinematic multilingual preloader.
- Journey timeline of education and experience.

## Tech Stack
- **Frontend**: HTML5, CSS3, JavaScript (Vanilla)
- **Icons**: Devicon
- **Design**: Glassmorphism, Responsive Grid
